Badges:

Para poder aplicar el estilo badge deberá hacer el llamado a la clase tal cual se llama el estilo. Este estilo tiene diferentes aplicaciones; se puede utilizar en encabezados, botones y otros elementos.

Ejemplo de badges en encabezados: 



    <h1>Esto es un encabezado. <span class="badge">New</span></h1>
    <h2>Esto es un encabezado. <span class="badge-bg-akio">New</span></h2>
    <h3>Esto es un encabezado. <span class="badge-bg-chihiro-yubaba">New</span></h3>
    <h4>Esto es un encabezado. <span class="badge-bg-haku-zeniba">New</span></h4>
